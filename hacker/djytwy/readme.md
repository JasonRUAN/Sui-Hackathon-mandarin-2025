## 其他附加说明
>> 主要的玩法:
1. free mode: 可以随意玩，game over 后没有奖励，没有 bouns。
2. PVP mode: 排行榜模式，玩家需支付一定的 token 购买门票，买门票的钱有两部分用处：
    1):会存入奖池，当天的某一个时间点给排行榜中排名高的玩家分钱。
    2):在游戏 game over 的时候给玩家分配一个随机的小于玩家付门票的 bouns。
其他:
    游戏中有商店，玩家可以通过商店购买皮肤。
