## 其他附加说明
* 平台提供积分，买家可以用积分购买上架的游戏资源。
* 作者上架创意想法，游戏资源,如动画，角色，地图，demo展示等。
* 作者可发行自己的积分币，用来将来给读者粉丝福利之用。
* 买家浏览上架的游戏资源，并可用积分投票，如果游戏最终上线，买家可以获得积分奖励和游戏优惠。
* 积分作为结算的依据，然后根据盈利比列进行发放。
